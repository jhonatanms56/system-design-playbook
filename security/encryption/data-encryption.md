# Data Encryption

Encryption ensures that data remains confidential and secure.

## Encryption in Transit

Protecting data as it moves over the network using protocols like TLS/SSL.

## Encryption at Rest

Protecting data stored on disks using methods like:
- Transparent Data Encryption (TDE)
- File-level encryption
- Cloud-provider managed keys (KMS)

## Secrets Management

Using dedicated tools like HashiCorp Vault, AWS Secrets Manager, or GCP Secret Manager to store and manage sensitive information.
